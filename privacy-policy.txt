WHATMOD PRIVACY POLICY

Last Updated: May 20, 2026

WhatMod ("the Extension") is designed to help Whatnot moderators manage moderation tools, messages, notes, reminders, and dashboard customization in a streamlined interface.

Your privacy is important. This Privacy Policy explains what information the extension collects, how it is used, and how it is stored.

1. Information We Collect

WhatMod does not collect, sell, transmit, or share personal information.

The extension may store user-created information locally within your browser, including:

• Custom messages
• Message banks
• Notes
• Reminder settings
• Dashboard preferences
• Appearance and customization settings
• Other extension configuration data

This information is stored locally on your device using Chrome extension storage.

2. How Information Is Used

Stored information is used only to:

• Save your extension settings
• Remember user preferences
• Provide moderation tools and customization features
• Improve the user experience within the extension

Information is not used for advertising, tracking, profiling, or analytics.

3. Data Sharing

WhatMod does not:

• Sell user information
• Share user information with third parties
• Transfer user information to external services
• Use user information for advertising purposes

4. Remote Services

WhatMod does not use remote servers to process user data.

All extension functionality operates locally within the browser unless future updates specifically state otherwise.

5. Clipboard Access

WhatMod may use clipboard functionality only when initiated by the user. Clipboard actions are used solely to copy moderation text or user-created content for convenience.

Clipboard contents are not stored, transmitted, monitored, or collected.

6. Website Access

WhatMod requests access only to supported Whatnot pages required for extension functionality.

Website access is used solely to display extension tools and provide moderation features.

7. Data Retention

Because user information is stored locally within the browser, data remains available until:

• The user deletes it
• Extension data is cleared
• The extension is removed

8. Children's Privacy

WhatMod is not intended for children under the age of 13 and does not knowingly collect personal information from children.

9. Changes to This Policy

This Privacy Policy may be updated periodically to reflect feature updates or legal requirements. Any changes will be reflected by updating the "Last Updated" date above.
