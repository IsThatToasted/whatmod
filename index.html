FAQ

Everything moderators want to know.

What is WhatMod?

WhatMod is a moderation toolkit built specifically for Whatnot moderators. It helps mods respond faster, manage giveaways, use reusable messages, and keep live-show workflows clean and professional.

Who is WhatMod made for?

WhatMod was designed for active Whatnot moderators who handle busy live chats, giveaways, sizing questions, announcements, rules reminders, and repetitive responses throughout streams.

How does WhatMod help during live shows?

WhatMod keeps your most-used messages and workflows in one fast-access workspace. Mods can quickly use prepared responses, manage show moments, and stay focused on chat.

What features are included?

WhatMod includes quick message banks, giveaway tools, editable cards, custom hotkeys, reusable workflows, saved preferences, and moderator-focused organization tools built for live selling environments.

Do I need technical experience to use it?

No. WhatMod is designed to be simple and fast to use during high-speed live streams. Most moderators can start using it immediately after setup.

How does access work?

WhatMod Pro is offered as 30-day membership access. During your active access period, you can use the latest available WhatMod build and receive supported updates.

Will new features be added?

Yes. WhatMod is actively evolving with planned improvements for smarter workflows, expanded moderation utilities, automation support, and stream-ready enhancements.

Is WhatMod affiliated with Whatnot?

No. WhatMod is an independent moderation tool created for moderators who work within the Whatnot ecosystem.
